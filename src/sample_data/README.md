# サンプルデータ

公開向けデモに差し替えやすいように、自作の 2D CAD / 3D CAD 入力をここで管理します。

## 方針

- 元ネタに依存しない自作サンプルだけを置く
- `DXF` と `STEP` を同じ寸法セットから生成する
- 部品ごとにメモと JSON 仕様も同梱する
- 後で見積デモへ差し替えるときは、ここを入力ソースにする

## 構成

- `generate_public_demo_parts.py`
  - サンプル部品を再生成するスクリプト
- `public_demo_parts/`
  - 生成済みの部品一式
- `public_demo_drawings/`
  - 図面画像の受け皿と図面仕様メモ

各部品ディレクトリには次を置きます。

- `*.dxf`
- `*.step`
- `notes_ja.md`
- `part_spec.json`

## 再生成

```bash
python3 src/sample_data/generate_public_demo_parts.py
```

## 利用上の注意

- このディレクトリのサンプルデータは、AI を使って生成した公開向け評価用サンプルです
- システム評価、デモ確認、画面比較、見積ロジック検証の用途では、必要に応じて自由に利用して構いません
- これらのデータは実在製品を正確に表したものではなく、製品としての整合性、加工成立性、寸法妥当性、量産適合性は保証しません
- 実製品の設計、製造、調達、品質保証には、そのまま使用しないでください
