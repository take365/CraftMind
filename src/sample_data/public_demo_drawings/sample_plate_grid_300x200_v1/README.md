# sample_plate_grid_300x200_v1

このフォルダは、公開向けデモ用の自作図面画像 `ベースプレート` を格納するための受け皿です。

## 判断

- この図面は公開デモ向けに使いやすい
- 形状が単純で、寸法と穴配列の説明が明確
- 現在の `sample_base_plate_a` とは寸法と穴数が一致しないため、別サンプルとして扱う

## 推奨ファイル名

- `sample_plate_grid_300x200_v1_drawing.png`

- 品名: ベースプレート
- 図番: BP-001
- 材質: SS400
- 板厚: t=12
- 外形: 300 x 200
- 穴: 15-φ10 貫通穴
- 穴配列: 5 列 x 3 段
- X 方向: 40, 55, 55, 55, 55, 40
- Y 方向: 40, 60, 60, 40
- 備考: 角部 C1 / バリ・カエリ除去
- 単位: mm
- 尺度: 1:2
